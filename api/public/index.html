API Method Listing

Public Methods

GET products/detail/:SKU

Returns detailed information regarding the product with the specified SKU.

GET products/search[?criteria1=a&criteria2=b]

Return an array of products matching the provided criteria.

GET products/options/:property

Returns all the possible listing options for a particular property, i.e., brand.

Secure Methods

POST products/adjust/quantity

Adjust the remaining quantity of items available. Use when a sale has been made.

POST Parameters

Parameter Name Description Example
token Your secure token "1234-5678"
SKU The SKU (Stock Keeping Unit) of the product you want to modify "abcd-efgh"
count The amount to adjust the product count by. -2
